# Todo APP
This is a basic todo application's backend developed with Nestjs


### Running Guide

step 1
```
npm install

```
step 2
```
cp .env.example .env
```
_**NB: Add env variables value**_

step 3
```
npm run migration:run
```

step 4
```
npm run start:dev
```

This will start the server on [127.0.0.1:3000](http://127.0.0.1:3000)
